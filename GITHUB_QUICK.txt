🚀 QUICKEST METHOD - Automatic Script

1. Create repository on GitHub:
   https://github.com/new
   - Name: tidbyt-matrix
   - Public (recommended)
   - Do NOT initialize with README
   → Click "Create repository"

2. Run the setup script:
   chmod +x setup_github.sh
   ./setup_github.sh

3. When prompted, enter:
   - GitHub username
   - Repository name (default: tidbyt-matrix)
   - Local project path (default: ~/tidbyt-matrix)
   - Path to files (outputs folder)

4. After script completes, run these commands:
   cd ~/tidbyt-matrix
   git commit -m "Initial commit: Tidbyt matrix display system"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/tidbyt-matrix.git
   git push -u origin main

🔧 MANUAL METHOD - Step by Step
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Create GitHub repository:
   https://github.com/new
   - Name: tidbyt-matrix
   - Public (optional)
   - Don't initialize with README
   → Copy the URL shown (looks like: https://github.com/YOUR_USERNAME/tidbyt-matrix.git)

2. Create local project directory:
   mkdir ~/tidbyt-matrix
   cd ~/tidbyt-matrix

3. Initialize git:
   git init

4. Copy all files from outputs folder:
   cp /path/to/outputs/tidbyt_*.py .
   cp /path/to/outputs/real_api_examples.py .
   cp /path/to/outputs/*.md .
   cp /path/to/outputs/*.service .
   cp /path/to/outputs/*.txt .

5. Create .gitignore:
   git add .gitignore

6. Stage files:
   git add .

7. Configure git (first time only):
   git config user.name "Your Name"
   git config user.email "your@email.com"

8. Commit:
   git commit -m "Initial commit: Tidbyt matrix display system"

9. Set branch name:
   git branch -M main

10. Add remote (replace YOUR_USERNAME):
    git remote add origin https://github.com/YOUR_USERNAME/tidbyt-matrix.git

11. Push to GitHub:
    git push -u origin main

✓ Done! Your code is on GitHub!

🔐 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"Permission denied (publickey)" error:
  → Check SSH key setup: https://docs.github.com/en/authentication/connecting-to-github-with-ssh
  → Or use HTTPS instead of SSH

"authentication failed" error:
  → Use Personal Access Token instead of password
  → Generate at: https://github.com/settings/tokens
  → When pushing, use token as password

"fatal: remote origin already exists":
  → git remote remove origin
  → git remote add origin https://github.com/YOUR_USERNAME/tidbyt-matrix.git

Git not configured:
  → git config --global user.name "Your Name"
  → git config --global user.email "your@email.com"
